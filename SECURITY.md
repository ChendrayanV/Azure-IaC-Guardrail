# Security Policy

Report vulnerabilities privately through [GitHub Security Advisories](https://github.com/ChendrayanV/Azure-IaC-Guardrail/security/advisories/new).

Do not create a public issue containing credentials, Terraform plans or state, tfvars, resource IDs, tenant IDs, subscription IDs, backend configuration, or sensitive logs.

The project supports the latest released version unless a release note states otherwise. Security fixes are prioritized by exploitability, impact, and exposure of local Terraform or Azure authentication data.
