# Azure IaC Guardrail

Azure IaC Guardrail is a Visual Studio Code extension that helps engineering
teams design, scan, review, and evidence Azure Terraform before deployment.

## Product promise

- Detect security and compliance risks while Terraform is being authored.
- Evaluate resolved plans without running `terraform apply`.
- Explain architecture risk, planned change impact, and remediation.
- Generate reviewable evidence for engineering and assurance teams.
- Explore Azure architecture visually through preview experiences.
