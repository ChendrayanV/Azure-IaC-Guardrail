# Feature Guide

| Feature | What it provides | Page |
|---|---|---|
| Static scanning | Offline, variable-aware Terraform checks | [Static Scanning](Static-Scanning) |
| Plan scanning | Resolved values, relationships, and plan-only checks | [Plan Scanning and Review](Plan-Scanning-and-Review) |

| Plan comparison | Added, removed, changed, and unchanged resource summary | [Plan Scanning and Review](Plan-Scanning-and-Review) |

| Editor experience | Diagnostics, hovers, completions, provenance, and quick fixes | [Editor Experience](Editor-Experience) |
| Workspace governance | Roots, regions, tags, exclusions, exceptions, and assumptions | [Workspace Governance](Workspace-Governance) |
| Resource Cost | Supported retail estimates and explicit assumptions | [Resource Cost](Resource-Cost) |
| Cloud Canvas | Static GraphViz Azure architecture diagrams from Terraform configuration | [Cloud Canvas](Cloud-Canvas) |
| Reporting | PDF, JSON, and Markdown evidence | [Evidence and Reporting](Evidence-and-Reporting) |

Resource Cost and Cloud Canvas are Preview features. Review their output before
using it in financial, architecture, or deployment decisions.
